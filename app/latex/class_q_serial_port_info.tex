\doxysubsection{Detailed Description}
Provides information about existing serial ports. 

\textbackslash{}inmodule Qt\+Serial\+Port \begin{DoxySince}{Since}
5.\+1
\end{DoxySince}
Use the static functions to generate a list of \mbox{\hyperlink{class_q_serial_port_info}{Q\+Serial\+Port\+Info}} objects. Each \mbox{\hyperlink{class_q_serial_port_info}{Q\+Serial\+Port\+Info}} object in the list represents a single serial port and can be queried for the port name, system location, description, and manufacturer. The \mbox{\hyperlink{class_q_serial_port_info}{Q\+Serial\+Port\+Info}} class can also be used as an input parameter for the set\+Port() method of the \mbox{\hyperlink{class_q_serial_port}{Q\+Serial\+Port}} class.
